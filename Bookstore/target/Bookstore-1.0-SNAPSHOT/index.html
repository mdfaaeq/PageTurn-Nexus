Welcome to Bookstore API

Below are all the available endpoints for the Bookstore REST API.

Books

GET http://localhost:8080/bookstore/api/books

Get all books

GET http://localhost:8080/bookstore/api/books/{id}

Get book by ID

POST http://localhost:8080/bookstore/api/books

Create a new book

PUT http://localhost:8080/bookstore/api/books/{id}

Update a book

DELETE http://localhost:8080/bookstore/api/books/{id}

Delete a book

Authors

GET http://localhost:8080/bookstore/api/authors

Get all authors

GET http://localhost:8080/bookstore/api/authors/{id}

Get author by ID

GET http://localhost:8080/bookstore/api/authors/{id}/books

Get books by author

POST http://localhost:8080/bookstore/api/authors

Create a new author

PUT http://localhost:8080/bookstore/api/authors/{id}

Update an author

DELETE http://localhost:8080/bookstore/api/authors/{id}

Delete an author

Customers

GET http://localhost:8080/bookstore/api/customers

Get all customers

GET http://localhost:8080/bookstore/api/customers/{id}

Get customer by ID

POST http://localhost:8080/bookstore/api/customers

Create a new customer

PUT http://localhost:8080/bookstore/api/customers/{id}

Update a customer

DELETE http://localhost:8080/bookstore/api/customers/{id}

Delete a customer

Shopping Cart

GET http://localhost:8080/bookstore/api/customers/{customerId}/cart

Get customer's cart

POST http://localhost:8080/bookstore/api/customers/{customerId}/cart/items

Add item to cart

PUT http://localhost:8080/bookstore/api/customers/{customerId}/cart/items/{bookId}

Update cart item quantity

DELETE http://localhost:8080/bookstore/api/customers/{customerId}/cart/items/{bookId}

Remove item from cart

Orders

GET http://localhost:8080/bookstore/api/customers/{customerId}/orders

Get all orders for a customer

GET http://localhost:8080/bookstore/api/customers/{customerId}/orders/{orderId}

Get specific order by ID

POST http://localhost:8080/bookstore/api/customers/{customerId}/orders

Create new order from customer's cart
